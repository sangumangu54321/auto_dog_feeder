# what is that?
This is a device which feed the dog automaticly or just with a one click with a help of mobile application

# feature 
- Wifi connectivite (use it from any conear of the world)
- Modile application (user friendly expreance)
- Scheduling even work offline (ensure the user of dog health) 
- really time food status bar (help the user to know does there dog eat the food)
- have a load cell (help the machine to give proper amount of food)
- backup battery (To ensure the device must be online 24/7 )
